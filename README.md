### Usage:

#### Installing Sass and Pug
<br>

Install sass in all your computer:

```
npm install sass -g
```

Or nnstall sass in the project:

```
npm install sass
```

Install Pug:

```
npm install pug-cli -g
```

#### Clone repository
<br>

Use `git clone` command from your CLI
```
git clone git@github.com:itsarreguin/elements.git
```
